Introduction

Module 1 focused on the programming language Logo, specifically its Turtle Graphics feature. Logo is an educational programming language created in 1967 by Wally Feurzeig, Seymour Papert, and Cynthia Solomon [1]. Seymour Papert designed Logo to enhance children's thinking and problem-solving abilities. A small mobile robot known as the "Logo Turtle" was developed to help children learn how to solve simple problems in a playful environment. One of the main objectives of the Logo Foundation research group is to strengthen the capacity for learning and knowledge acquisition [2].
In the lesson, we learn how to:
Initialize a turtle.
Set the turtle's color and background.
Move the turtle to a suitable starting point.
Draw a simple square.
Display the drawing.
The following code is used to initialize the turtle environment:
- `initializeTurtle()` sets up the turtle's enviroment.
- `showturtle()` makes the turtle visible.
- `color('blue')` changes the pen color to blue.
- `bgcolor('white')` sets the background color to white.
